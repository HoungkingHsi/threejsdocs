包围盒辅助（[name]）

用于创建对象和世界轴对齐的包围盒的一个帮助对象。

例子

var hex = 0xff0000; var sphereMaterial = new THREE.MeshLambertMaterial( {color: 0x00ff00} ); var sphere = new THREE.Mesh( new THREE.SphereGeometry( 30, 12, 12), sphereMaterial ); scene.add( sphere ); var bbox = new THREE.BoundingBoxHelper( sphere, hex ); bbox.update(); scene.add( bbox );
注意，这个辅助对象将创建一个带 [page:盒子模型(BoxGeometry)] 的线框 [page:网孔(Mesh)]对象; 这个包围盒将因此具有面对角线。你可能想用[page:BoxHelper], 该对象生成一个不包含面对角线的 [page:Line] 对象。

构造器（Constructor）

[name]([page:3D对象(Object3D) object], [page:Number hex])

object -- Object3D -- 需要显示和世界轴对齐的包围盒的对象。
hex -- 16进制颜色值，ex:0x888888
这将创建一个Line类型的包围盒对象。

属性（Properties）

[property:Object3D object]

需要显示和世界轴对齐的包围盒的对象。

[property:Box3 box]

包围盒对象。

方法（Methods）

[method:null update]()

基于对象属性更新BoundingBoxHelper。
